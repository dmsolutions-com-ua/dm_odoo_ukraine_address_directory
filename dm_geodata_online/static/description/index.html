Адресний довідник України — повний набір

Парасольковий модуль: одним встановленням підтягує ядро та всі інтеграції автозаповнення українських адрес.

Поставте цей модуль, щоб одразу отримати Geodata.online у всій системі. Він залежить від ядра dm_geodata_connector, а тонкі бриджі (контакти, CRM, компанії, кадри, банки) самостійно вмикаються там, де встановлено відповідний застосунок.

Видалення цього модуля прибирає бриджі; ядро лишається — його видаляють окремо. Налаштування виконуються у ядрі: Налаштування → Geodata.online → Облікові дані API.
